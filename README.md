## 🚀 Quick Start Guide

### 📦 Installation

```bash
# Clone the repository
git clone https://github.com/minindu-alwis/ExploreSriLanka-TravelLK.git

# Navigate to the project directory
cd ExploreSriLanka-TravelLK

# Launch with live server (recommended)
# Using VS Code Live Server extension or:
npx http-server . -p 8080

# Open your browser to localhost:8080
```

## 🌍 Comprehensive Coverage

<details>
<summary><h3>🏖️ Western Province</h3></summary>

**Commercial Heart of Sri Lanka**
- **Colombo** - Bustling capital with modern skylines and colonial charm
- **Gampaha** - Gateway to pristine beaches and cultural heritage
- **Kalutara** - Coastal paradise with ancient Buddhist temples

*Featured attractions: Galle Face Green, Gangaramaya Temple, Kelaniya Raja Maha Vihara*

</details>

<details>
<summary><h3>🌸 Central Province</h3></summary>

**Cultural and Scenic Highlands**
- **Kandy** - Sacred city housing the Temple of the Sacred Tooth Relic
- **Matale** - Aromatic spice gardens and rolling hill country
- **Nuwara Eliya** - "Little England" with colonial architecture and tea estates

*Featured attractions: Temple of the Tooth, Royal Botanical Gardens, Tea Museum*

</details>

<details>
<summary><h3>🏖️ Southern Province</h3></summary>

**Coastal Beauty and Colonial Heritage**
- **Galle** - UNESCO World Heritage fort city with Dutch architecture
- **Matara** - Pristine beaches and historic Buddhist temples
- **Hambantota** - Wildlife sanctuaries and modern infrastructure

*Featured attractions: Galle Fort, Mirissa Beach, Yala National Park*

</details>

<details>
<summary><h3>🌾 Complete Island Coverage</h3></summary>

**All 9 Provinces Included**
- Northern Province - Ancient capitals and cultural significance
- Eastern Province - Unspoiled beaches and diverse wildlife
- North Western Province - Historical sites and fishing communities
- North Central Province - Ancient ruins and archaeological wonders
- Uva Province - Mountain peaks and pristine wilderness
- Sabaragamuwa Province - Gem mining heritage and natural beauty

*Over 200+ destinations, 500+ hotels, and countless cultural experiences*

</details>

We welcome passionate contributors who want to help showcase Sri Lanka's incredible beauty and rich culture!

### 🚀 How to Contribute

1. **Fork the repository** and create your feature branch
2. **Commit your changes** with descriptive messages
3. **Push to your branch** and create a Pull Request
