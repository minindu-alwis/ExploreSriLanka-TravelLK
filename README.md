## 🚀🚀🚀🚀 **Quick Start Guide**

### 🔧 **Installation & Setup**

```bash
# Clone the repository
git clone https://github.com/minindu-alwis/ExploreSriLanka-TravelLK.git

# Navigate to project directory
cd ExploreSriLanka-TravelLK

# Open with live server (VS Code extension recommended)
# Or simply open index.html in your browser

# For development server (if using Node.js)
npx http-server . -p 8080
```

### 📁 **Project Structure**

```
ExploreSriLanka-TravelLK/
├── 🏠 index.html              # Main landing page
├── 🗂️ assets/
```
